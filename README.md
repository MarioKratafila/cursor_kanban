# Kanban Project Manager

A single-board Kanban web app built with Next.js.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tests

```bash
cd frontend
npm test          # unit tests (Vitest)
npm run test:e2e  # end-to-end tests (Playwright)
```

## Build

```bash
cd frontend
npm run build
npm start
```
